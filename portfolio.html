# 연스타그램(토이프로젝트)

개발인원: 총 2명
프론트엔드 개발자 1명과 총 2명이서 진행한 프로젝트
서버는 node.js Express를 사용했고
프론트는 vue.js를 사용해서 만든 인스타그램 클론 어플리케이션
퍼블리셔에서 프론트엔드개발자로 넘어가는과정에서 만든 프로젝트로
개인적으로 소중했던 프로젝트
https://github.com/bdabin/yeonstargram-spa (opens new window)
